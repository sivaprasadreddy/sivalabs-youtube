# SivaLabs YouTube Videos Explorer
Go to [SivaLabs YouTube Videos](https://sivaprasadreddy.github.io/sivalabs-youtube/) `docs/index.html` in your browser to explore the videos available on [SivaLabs](https://www.youtube.com/@sivalabs/videos) YouTube channel.

## Development Setup
Create a YouTube Data API key by following the instructions [here](https://developers.google.com/youtube/v3/getting-started).

## Usage

1. Set the `YOUTUBE_DATA_API_KEY` environment variable
2. Run `Main.java`
